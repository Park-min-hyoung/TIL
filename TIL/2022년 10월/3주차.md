## 배운 것

### `스포츠 용병 서비스 팀플`

> onclick시 호출 된 함수가 event 및 다른 파라미터를 전달 받을 경우

- onClick시 event 객체만 사용할 때 처럼 전달인자로 event 관련해서 아무것도 안 넘겨 주어도 자동으로 event 객체를 이용할 수 있다고 생각
- event를 콘솔로 출력 해봤지만 undefined 출력
- event 객체 외에 다른 파라미터를 사용하는 경우에는 event와 관련된 객체를 전달해줘야 함

```jsx
const selectItem = async (index, event) => {};

// 오류 발생 코드
<Item key={item} onClick={() => selectItem(idx)}></Item>
// 오류 해결 코드
<Item key={item} onClick={(e) => selectItem(idx, e)}></Item>
```
